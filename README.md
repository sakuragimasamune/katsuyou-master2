# 活用マスター 第二章

古文動詞の **「行・活用の種類・活用形」** を三段階で判定する高校古文学習アプリ。
桜木マサムネ先生のビジュアルノベル形式の講義と、即時フィードバック付き練習問題。

> 第一章「活用の種類判別」を終えた学習者向けの続編です。

## 特長

- **VN形式の講義** — 1問ずつ手順を踏ませて、解答後に「解説を聞く」で桜木節の解説と推論チェーンを表示
- **三段階判定** — 行 → 活用の種類 → 活用形 の順で部分点が見える
- **接続のショートカット** — 「て／けり／たり」「む／ざ」「ども／ば」などの接続助詞ルールを段階的に導入
- **２択ひっかけ対応** — 第二部では「とて＋ぞ」の係り結び・「ば」の上は未然形か已然形・体言の上は連体形 など上級テクを扱う
- **PWA対応** — オフラインで動く・ホーム画面に追加可能・Chromebook / Android / iPhone 対応
- **シングルファイル設計** — `index.html` だけで動作（外部依存は Google Fonts のみ）

## ファイル構成

```

```


### キャッシュの更新について

Service Worker は `sw.js` の `CACHE_VERSION` 文字列をキーにキャッシュを管理しています。
コンテンツを更新したら `sw.js` の `const CACHE_VERSION = 'katsuyou-ch2-v1';` の `v1` を `v2` などに上げると、次回アクセス時に旧キャッシュが破棄されて新版が取得されます。

## 進捗保存

- 進捗は `localStorage` に `katsuyou_ch2_u` キーで保存されます
- 第一章アプリの `katsuyou_u` とは独立しているので、両方併用しても干渉しません
- メニュー右上の「🔒 リセット」で進捗を初期状態に戻せます
- 「🔓 リピーター用ロック解除」で全コンテンツを即時開放できます

## アンロックの流れ

```
講義1（最初から開放）
  ├─ 練習1-1 を開放
  └─ 練習1-2 を開放
       └─（70% 以上で）講義2 を開放
              ├─ 練習2-1 を開放
              └─ 練習2-2 を開放
                   └─（90% 以上で）「第二章 完全制覇」バッジ
```
